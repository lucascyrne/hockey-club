# Logo "Hockey Club" — prompt para ChatGPT / DALL·E

Gere um **PNG com fundo transparente** e guarde como:

`public/textures/hockey-club-logo.webp`

Recomendado: **1024×1024 px**, área útil centralizada, margem ~10% nas bordas.

---

## Prompt (copiar)

```
Game logo emblem for "HOCKEY CLUB", transparent background WEBP, no background color, no floor shadow.

Main symbol: air hockey puck as a fiery comet — bright yellow-gold disc core (#ffea00) with orange-red flame trail behind it (motion toward upper-left), controlled arcade flames not realistic explosion, clean silhouette readable at small size.

Below the puck emblem, bold wordmark text: "HOCKEY CLUB" in two lines or single line, uppercase, modern geometric sans-serif, white letters with thin cyan neon outline glow (#00e5ff), subtle pink accent (#ff2d6a) on "CLUB" or underline only.

Style: vector arcade neon, Tron / retro-futuristic sports logo, high contrast, crisp edges, suitable for overlay on dark blue game table, no photorealism, no 3D render perspective, flat sticker style.

Constraints: centered composition, square canvas, transparent pixels outside logo, no table, no paddles, no extra text, no watermark, no mockup, no circular badge border unless very subtle.
```

## Prompt negativo (se disponível)

```
background, solid color backdrop, room, table surface, perspective, blurry, muddy colors, realistic fire photo, skull, hockey stick, ice rink
```

## Após gerar

1. Recortar fundo transparente se necessário.
2. Exportar PNG 1024×1024.
3. Salvar em `public/textures/hockey-club-logo.webp` (ou `.png`).
4. Reiniciar `npm run dev` — a logo aparece ao centro da mesa automaticamente.
